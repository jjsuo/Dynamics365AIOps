## 四象限决策

| 象限 | 判定标准 | 本需求是否落此 | 工作量/风险 |
|---|---|---|---|
| **标准(Standard)** | 开箱即用，零配置 | ☐ | 最低 |
| **配置(Configure)** | 用表单/视图/BPF/business rule/flow 等低代码实现 | ☐ | 低，可维护 |
| **定制(Customize)** | 需 plugin / PCF / JS / Custom API | ☐ | 高，需 ALM + 测试 |
| **集成(Integrate)** | 需对接外部系统 | ☐ | 高，依赖外部 |

## 决策结论
- **选定象限**：配置
- **理由**：到期提醒可用 Power Automate 定时流 + business rule 实现，无需 pro-code
- **涉及 DKF 已有资产**：指向 `.dkf/skills/合同管理.md`（复用已有 new_contract 字段）
- **风险/前置依赖**：需确认提醒渠道（见 open_questions）
